## Features

- Single `Landing` Page
- `Fiverr` Profile Link
- `Upwork` Profile Link
- `Resume` Link
- Mention `Company`
- Recent Projects with `Live Link` & `Source Code`
- Fully responsive for `PC`, `Laptop`, `Tablet` & `Mobile`

## Tech Stack

- Framework: `Next.JS`
- Styling: `Tailwind CSS`
- Icons: `React Icons`
- Linting: `ESLint`
- Formatter: `Prettier`
